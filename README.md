**Corona Admin** is a highly responsive, deeply dark-themed administrative dashboard template built on top of the Bootstrap framework. It is designed to be visually striking and easy on the eyes, making it the perfect starting point for complex web applications, CMS backends, crypto panels, and data analytics dashboards.

[👉 **View Live Demo Here**](https://smaron47.github.io/corona-bootstrap-dark-admin-dashboard/) 

## ✨ Key Features

* **Sleek Dark Mode UI:** A beautifully crafted dark theme with high-contrast gradient accents and clean typography.
* **Rich Data Visualization:** Integrated with **Chart.js** for rendering interactive transaction histories and revenue charts.
* **Global Analytics:** Includes **jVectorMap** for visualizing audience and visitor demographics by country.
* **Advanced Components:** Features an **Owl Carousel** for portfolio slides, a dynamic To-Do list, and interactive progress bars.
* **Fully Responsive:** Adapts seamlessly to desktop, tablet, and mobile screens with a collapsible off-canvas sidebar.
* **Pre-built Pages:** Includes layouts for basic UI elements, forms, tables, login/register screens, and error pages.

## 🛠️ Technology Stack & Plugins

* **HTML5 / CSS3**
* **Bootstrap** (Core framework)
* **Material Design Icons (MDI)** (Clean, modern iconography)
* **jQuery** (DOM manipulation)
* **Chart.js** (Data charts)
* **jVectorMap** (Interactive maps)
* **Owl Carousel 2** (Touch-enabled sliders)

## 🚀 How to Use / Installation

Because this is a static HTML/CSS/JS template, no complex local server setup is required to view the UI.

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/Smaron47/corona-bootstrap-dark-admin-dashboard.git](https://github.com/Smaron47/corona-bootstrap-dark-admin-dashboard.git)
    ```
2.  **Open locally:** Double-click the `index.html` file inside the downloaded folder to launch the dashboard in your browser.
3.  **Customize:** Edit the HTML files to hook up your own backend logic or replace the static chart data with live API feeds.
